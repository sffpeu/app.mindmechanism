# Mind Mechanism (M13)

Next.js app for **focused practice** with nine celestial mandala clocks: Firebase auth, timed sessions, notes, glossary, and lobby groups.

## MVP scope (what ships end-to-end)

1. **Sign in** — Email/password, Google, or email link (`/home`).
2. **Home mandala** — Layered clocks at `/layers` (dock: Home) with optional getting-started banner.
3. **Sessions** — Pick a clock, duration, and intention words; opens the live clock experience.
4. **Dashboard** — Profile, time stats, recent sessions, group session summary, practice shortcuts.
5. **Glossary & notes** — Vocabulary search and session notes.
6. **Lobby** — Group scheduling and gathering flows (where configured).

Production: **app-mindmechanism.vercel.app**. Repo: `sffpeu/app.mindmechanism` on GitHub.
